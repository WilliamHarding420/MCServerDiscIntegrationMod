

Neoforge mod that connects to a discord bot (defined in mod config) and sends discord messages into server chat and messages from defined discord channels to minecraft chat.   


You must setup your own discord bot with the Message Content intent, and being able to view the channels you want it to read messages from (must have roles to send/receive messages from the channels).  



Install the mod into a 1.21.1 NeoForge 21.1.233 server (don't install in client) and run the server to generate config, then modify config adding the required values for the bot to run. (To get a channel ID, just right click the channel and click "Copy Channel ID")



Uses [JDA](https://github.com/discord-jda/JDA) for Discord Integration.
